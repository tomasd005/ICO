# Code of Conduct

## Our Standard

Contributors are expected to:

- Be respectful and constructive
- Focus on technical merit and collaboration
- Avoid harassment, discrimination, and personal attacks

## Scope

This applies to issues, pull requests, discussions, and other project spaces.

## Enforcement

Project maintainers may edit, remove, or reject contributions and interactions that violate this code.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainer.
