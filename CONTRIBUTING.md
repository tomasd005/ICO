# Contributing

Thanks for contributing.

## Prerequisites

- Node.js 20+
- npm 9+

## Local Setup

```bash
npm install
npm run compile
npm test
```

## Quality Checks

```bash
npm run lint
npm run coverage
```

## Pull Requests

- Keep PRs focused and small.
- Add or update tests for behavior changes.
- Update `README.md` and `CHANGELOG.md` when relevant.
- Use clear commit messages.

## Commit Style

Examples:

- `feat: add whitelist checks for token campaigns`
- `fix: prevent ico token over-allocation`
- `docs: expand security assumptions`
